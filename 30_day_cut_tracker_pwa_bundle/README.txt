30 Day Cut Tracker PWA

Files:
- index.html
- manifest.json
- sw.js
- icons/

How to use on iPhone:
1. Put these files on a web host. Easiest options:
   - GitHub Pages
   - Netlify
   - Vercel
2. Open the hosted URL in Safari.
3. Tap Share > Add to Home Screen.

Important:
- On iPhone, a true installable web app works best from a hosted https URL, not directly from a local file.
- The tracker stores data in the browser on that device.
